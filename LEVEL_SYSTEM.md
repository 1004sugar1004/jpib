# 🗺️ IB 탐험가 레벨 시스템 가이드

본 문서는 'IB 탐험가' 애플리케이션의 10단계 레벨 시스템과 각 단계별 상세 정보를 안내합니다.

## 1. 레벨 구성 및 획득 조건

학생들이 활동을 통해 얻은 경험치(XP)에 따라 총 10단계의 등급이 부여됩니다.

| 단계 | 등급 이름 | 필요 XP | 설명 |
| :--- | :--- | :--- | :--- |
| **Lv.1** | 초보 탐험가 | 0+ | 이제 막 탐험을 시작한 새싹입니다. |
| **Lv.2** | 견습 탐험가 | 200+ | 본격적인 탐험을 시작했습니다. |
| **Lv.3** | 숙련된 탐험가 | 600+ | 탐험의 재미를 알아가는 중입니다. |
| **Lv.4** | 브론즈 탐험가 | 1,200+ | 기초를 탄탄히 다진 탐험가입니다. |
| **Lv.5** | 실버 탐험가 | 2,000+ | 안정적인 실력을 갖춘 탐험가입니다. |
| **Lv.6** | 골드 탐험가 | 3,000+ | 황금빛 미래를 향해 나아가는 탐험가입니다. |
| **Lv.7** | 플래티넘 탐험가 | 4,500+ | 수많은 역경을 이겨낸 베테랑입니다. |
| **Lv.8** | 다이아몬드 탐험가 | 6,000+ | 보석처럼 빛나는 지혜를 가졌습니다. |
| **Lv.9** | 그랜드 마스터 탐험가 | 8,000+ | 탐험의 진리를 깨달은 마스터입니다. |
| **Lv.10** | 전설의 마스터 탐험가 | 10,000+ | 모든 지식을 섭렵한 최고의 탐험가입니다. |

## 2. 필요한 이미지 자산 (Asset List)

각 단계의 시각적 변화를 위해 아래와 같은 컨셉의 이미지가 필요합니다.
이미지는 배경이 투명한 **PNG** 형식을 권장하며, 크기는 **512x512px** 이상이 적당합니다.

### 캐릭터 이미지 (ASSETS.characters)
- `explorer_lv1.png`: 새싹, 작은 가방을 멘 아이
- `explorer_lv2.png`: 돋보기를 든 아이
- `explorer_lv3.png`: 지도를 든 아이
- `explorer_lv4.png`: 구리색 메달이나 장비를 갖춘 모습
- `explorer_lv5.png`: 은색 장비, 전문적인 탐험가
- `explorer_lv6.png`: 황금색 장비, 빛나는 망원경
- `explorer_lv7.png`: 백금색 장비, 고급 나침반
- `explorer_lv8.png`: 보석이 박힌 장비, 빛나는 아우라
- `explorer_lv9.png`: 화려한 망토, 지혜로운 스승의 느낌
- `explorer_lv10.png`: 왕관, 전설적인 보물을 가진 최종 형태

## 3. 포인트(XP) 획득 가이드

- **지식 탐험**: +30 XP
- **플래티넘 카드**: +150 XP
- **기억력 강화**: +50 XP
- **퀴즈 챌린지**: +50 XP
- **음악 퀴즈**: +50 XP (정답당)
